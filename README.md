
📊 Project Overview
This interactive Superstore Sales & Profit Insights Dashboard provides comprehensive business intelligence for retail operations. It analyzes sales performance, profitability trends, customer segmentation, and geographical distribution to enable data-driven decision making.
Analysis Period: 2010-2013
Data Points: 9,426 orders across multiple product categories and customer segments

Order Details: Order ID, Order Date, Ship Date, Order Priority
Customer Information: Customer Name, Segment, Region, State, City
Product Data: Category, Sub-Category, Product Name
Financial Metrics: Sales, Profit, Discount, Quantity
